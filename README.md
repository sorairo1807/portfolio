# マーケティング導線設計ポートフォリオ

静的HTML/CSS/JavaScriptで作成した1ページのポートフォリオサイトです。

## 公開前に差し替えるもの

- 問い合わせ先を変更する場合は `index.html` の `mailto:kkawanabe@srir.jp`
- 必要に応じて `<title>`、OGP文言、フッター表記

## 公開方法

このフォルダの `index.html`、`styles.css`、`script.js`、`assets/` をそのまま静的ホスティングにアップロードできます。Netlify、Vercel、GitHub Pages などで公開できます。
